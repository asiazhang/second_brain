## C++项目

Impeller是一个C++项目。之前是采用的bat文件调用IDE的命令行直接进行构建，不透明。方式如下：

```@echo off

call "E:\Program Files (x86)\Microsoft Visual Studio 9.0\VC\bin\vcvars32.bat"
echo Current directory is:
cd
cd /d %~dp0
DEVENV /REBUILD "Release" "Impeller.sln"
```
